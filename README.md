## Features
* Terminal based
* Screen size maps


## Get started
### Requirements
Please make sure your operating system has the following software installed.
* [Bundler](https://bundler.io/)

### Install
run `bundle install` to install the dependencies.

### Play
run `rake start` to start playing.
